# STMS
Density based Smart Traffic Management System;

The primary objective of this project is to design and develop a
 simulated Smart Traffic Management System that dynamicaly
 manages urban traffic conditions to improve flow efficiency, reduce
 congestion, and enhance road safety. The system aims to emulate real
time traffic behavior and adapt signal timings using Python-based
 modules.
 This simulation wil be built using Python programming language,
 integrating multiple modules such as pygame for graphical rendering,
 time and threading for real-time control, and random for simulating
 unpredictable traffic behavior
